# cache テーブル

## 概要

Laravelのキャッシュシステムで使用するテーブル。

## テーブル構造

| カラム名   | データ型   | NULL | デフォルト値 | 制約        | 説明                         |
| ---------- | ---------- | ---- | ------------ | ----------- | ---------------------------- |
| key        | VARCHAR    | NO   | -            | PRIMARY KEY | キャッシュキー               |
| value      | MEDIUMTEXT | NO   | -            | -           | キャッシュ値                 |
| expiration | INTEGER    | NO   | -            | -           | 有効期限(UNIXタイムスタンプ) |

## インデックス

| インデックス名 | カラム | タイプ      |
| -------------- | ------ | ----------- |
| PRIMARY        | key    | PRIMARY KEY |

## 備考

- Laravelのキャッシュドライバーが 'database' の場合に使用
- システムが自動的に管理するテーブル
