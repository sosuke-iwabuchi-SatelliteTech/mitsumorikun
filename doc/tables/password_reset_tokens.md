# password_reset_tokens テーブル

## 概要

パスワードリセットトークンを管理するテーブル。

## テーブル構造

| カラム名   | データ型  | NULL | デフォルト値 | 制約        | 説明             |
| ---------- | --------- | ---- | ------------ | ----------- | ---------------- |
| email      | VARCHAR   | NO   | -            | PRIMARY KEY | メールアドレス   |
| token      | VARCHAR   | NO   | -            | -           | リセットトークン |
| created_at | TIMESTAMP | YES  | NULL         | -           | 作成日時         |

## インデックス

| インデックス名 | カラム | タイプ      |
| -------------- | ------ | ----------- |
| PRIMARY        | email  | PRIMARY KEY |

## 備考

- パスワードリセット機能で使用
- トークンは一定時間後に無効化される
- システムが自動的に管理するテーブル
