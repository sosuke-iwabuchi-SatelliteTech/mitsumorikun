# cache_locks テーブル

## 概要

Laravelのキャッシュロック機能で使用するテーブル。

## テーブル構造

| カラム名   | データ型 | NULL | デフォルト値 | 制約        | 説明                         |
| ---------- | -------- | ---- | ------------ | ----------- | ---------------------------- |
| key        | VARCHAR  | NO   | -            | PRIMARY KEY | ロックキー                   |
| owner      | VARCHAR  | NO   | -            | -           | ロック所有者                 |
| expiration | INTEGER  | NO   | -            | -           | 有効期限(UNIXタイムスタンプ) |

## インデックス

| インデックス名 | カラム | タイプ      |
| -------------- | ------ | ----------- |
| PRIMARY        | key    | PRIMARY KEY |

## 備考

- Laravelのキャッシュロック機能で使用
- 並行処理制御のために使用
- システムが自動的に管理するテーブル
