# 見積テンプレート機能

## 概要
見積テンプレート機能は、見積書を作成する際に使用するテンプレートを管理する機能です。

## 目的
見積書の作成を効率化し、見積書の品質を向上させる。

## 保存項目
見積もりの各行（項目）を保存できるようにする。
例えば、片付け作業という見積については
運搬作業、処分費用、諸経費などの項目がテンプレートとして保存するような運用を想定している。

## 機能
- 見積テンプレートの作成
- 見積テンプレートの編集
- 見積テンプレートの削除
- 見積テンプレートの適用

## 画面
- 見積テンプレート一覧画面
- 見積テンプレート作成画面
- 見積テンプレート編集画面

- 見積作成画面（既存）
    - テンプレートを適用する機能を追加
    - テンプレートを保存する機能を追加（作成中の見積もりをテンプレートへコピーできるイメージ

## データベース
- estimate_templates テーブル

## API
- GET /api/estimate-templates
- POST /api/estimate-templates
- GET /api/estimate-templates/{id}
- PUT /api/estimate-templates/{id}
- DELETE /api/estimate-templates/{id}
