# 見積

主に工事の見積もり等を想定している。

## 主な項目

- 顧客(顧客マスタを参照)
- タイトル（例：〇〇様邸工事）
- 見積バージョン（変更履歴に近い。バージョンを管理する。）
- 見積番号
- 見積日（見積バージョンの作成日）
- 受け渡し期日
- 住所（工事場所）
- 支払い条件
- 有効期限
- 見積項目（リスト。見積項目マスタを参照）
    - 見積項目はグルーピングの要素もある。例：現場作業（内訳として、搬出、搬入、解体など）

## 見積項目

搬出など、工事に関連する項目を管理する。標準単価や単位を管理する。
実際の見積もり作成時には、標準単価を参考として単価が前後する運用がある。
あくまで標準的なものであり、個別に変更することができる。
請求する項目がほとんどだが、引き取りなどでマイナスになるケースも存在する。

### 主な項目

- 名称（搬出、搬入、解体などの名称）
- 数量
- 単価
- 単位（m3など）
- 税区分（税込み、税抜き）
- 税率
- 合計金額
- 備考

# 顧客マスタ

見積の発行先、請求書の送付先。

## 主な項目

- 名称
- 住所
- 電話番号
- FAX番号
- メールアドレス
- 備考
