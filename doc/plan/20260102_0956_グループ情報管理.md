# グループに関する情報の管理機能

## 画面構成

- 会社情報管理
    - 見積/請求書記載事項（旧：基本情報）
    - 口座情報

## データ

- 見積/請求書記載事項
    - 会社名（見積・請求書記載用）
    - Invoice登録番号
    - 郵便番号
    - 住所1
    - 住所2
    - 電話番号
    - FAX番号
    - Eメール

- 口座情報
    - 表示選択（ラジオボタン：銀行 / ゆうちょ銀行）
    - ※表示しない選択肢は廃止。必ずどちらか一方が選択された状態とする。
    - 保存時に選択されていない方の情報はDB上でクリア（NULL化）する
    - 銀行の場合
        - 金融機関名
        - 支店名
        - 口座種別（普通/当座）
        - 口座番号
        - 口座名義
    - ゆうちょ銀行の場合
        - 記号（５桁）
        - 番号（最大８桁）
        - 口座名義

## 実装上の考慮事項

- 保存アクションを「見積/請求書記載事項」と「口座情報」で別々にする（各タブ内に保存ボタン）。
